## Huffman Compressor

Um compressor de arquivos *lossless* (sem perda de dados) construído do zero em Java, utilizando o clássico algoritmo de Codificação de Huffman.

## Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de aprofundar conhecimentos em Estruturas de Dados, Algoritmos e manipulação de memória em baixo nível (bits e bytes). Ele lê qualquer tipo de arquivo (texto, imagem, binário), mapeia a frequência de cada byte, constrói uma Árvore de Huffman e empacota os dados gerando um arquivo final comprimido.

O objetivo aqui não é competir com compressores comerciais modernos (que usam combinações de LZ77 com Huffman), mas sim demonstrar o domínio sobre os fundamentos da ciência da computação.
